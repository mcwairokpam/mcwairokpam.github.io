# Datasets

This folder contains datasets related to economic and financial indicators.

## Consumer Price Index (CPI)

File: CPIndex_Jan14-To-Jan25.csv  

Base Year: **2012 = 100**

Columns:

- Year – Year of observation
- Month – Month
- State – Geographic region
- Group – Major CPI group
- Sub Group – CPI subgroup
- Description – Commodity description
- Rural – CPI inflation rural
- Urban – CPI inflation urban
- Combined – CPI inflation combined
- Status – F = Final, P = Provisional
